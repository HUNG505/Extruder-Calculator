# **電纜押出製程智慧計算器 (V1.1)**

這是一個專為電纜製程設計的網頁應用程式 (PWA)，支援離線使用與行動裝置操作。  
供業務代表與製程工程師快速計算押出量、線速度與確認機台產能極限。

## **功能特色 (V1.1)**

* **動態機台產能匹配**：根據選擇的材料 (PVC, XLPE, LSHF) 自動切換機台的最大吐出量限制。  
* **雙向計算模式**：  
  1. 輸入線速度 \-\> 計算押出量 (kg/hr)。  
  2. 輸入目標產量 \-\> 反推建議線速度 (m/min)。  
* **即時防呆**：  
  * 自動偵測內徑是否大於外徑。  
  * 當計算結果超出機台極限時，顯示紅色警告。  
* **資料庫整合**：內建 PRD V1.1 規範之材料密度與機台規格表。

## **如何使用**

點擊右側的 GitHub Pages 連結即可直接開啟計算機。
